# Administrator Images

This directory contains images for the college administrators displayed on the administration page.

## Required Images

Please place the following images in this directory:

1. **dean.jpg** - College Dean photo
2. **academic-vice-dean.jpg** - Academic Vice Dean photo
3. **pgrcs-vice-dean.jpg** - PGRCS Vice Dean photo
4. **registrar.jpg** - Registrar photo
5. **laboratory-manager.jpg** - Laboratory Manager photo
6. **academic-council.jpg** - Academic Council photo (can be a group photo or representative image)

## Image Specifications

- **Format**: JPG, PNG, or WebP
- **Recommended size**: 800x800 pixels (square)
- **Aspect ratio**: 1:1 (square) for best results
- **File size**: Keep under 500KB for optimal loading

## Notes

- Images will be displayed at 192x192 pixels (12rem x 12rem) with rounded corners
- The component uses Next.js Image optimization for best performance
- Images should be professional headshots or official photos
